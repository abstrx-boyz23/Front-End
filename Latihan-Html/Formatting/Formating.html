Macam-Macam Formatting Pada HTML

- Text Tebal (Bold)

Ini Contoh Bold! Rasanya tidak menarik jika ada bagian penting tanpa diberi Bold

- Text Miring (Italic)

Ini Contoh Italic, jadi terlihat lebih menarik bukan?

- Text Garis Bawah (Underline)

Ini Contoh Underline. Yahaha Hayukkkk Ngoding Bro..

- Text Dicoret (Strikethrough)

Selesaikan masalahmu kawan, jangan pernah lari dari masalah. Tadi merupakan contoh text dicoret.

- Text Berpangkat (Superscript dan Subscript) 

• Fungsi Superscript adalah untuk membuat pangkat yg diatas.
Contoh nya seperti : Ek = 1/2 m.v2
• Fungsi Subscript adalah untuk membuat pangkat yang dibawah.
Contoh nya seperti : H2SO4
