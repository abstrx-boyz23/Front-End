Tutorial Heading Pakai CSS di HTML

Heading adalah judul sebuah artikel dan bagian dari artikel. Ada Enam level heading pada HTML, yakni dimulai dari H1, H2, H3, sampai H6.

ini heading huruf besar semua

Atribut style (text-transform: uppercase) berfungsi untuk mengubah satu kalimat menggunakan huruf besar semua.

ini heading huruf besar di awal kata

Atribut style (text-transform: capitalize) berfungsi untuk mengubah satu kalimat menjadi huruf besar di awal kata.

ini heading dengan warna aqua

Atribut style (color: aqua) berfungsi untuk mengubah warna font .

ini heading dengan warna dan background

Atribut style (color dan background-color) berfungsi untuk mengubah warna font dan juga background font.
